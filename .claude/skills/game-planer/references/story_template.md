# Story模板示例

## 示例1：基础移动功能

US_20251103121531_01
基础移动控制

作为 玩家
我想要 使用键盘或手柄控制角色在2D场景中移动
以便 探索游戏世界

验收标准（AC）
1. 玩家可以使用WASD或方向键控制角色八方向移动
2. 角色移动速度为5米/秒
3. 角色面对移动方向
4. 角色不能穿过障碍物
5. 手柄左摇杆可以控制角色移动

备注
- 颗粒度：1人日
- 技术债标签：
- 美术需求：PX（角色精灵图）

## 示例2：跳跃系统

US_20251103121531_02
跳跃机制

作为 玩家
我想要 按下空格键时让角色跳跃
以便 跨越障碍和到达高处平台

验收标准（AC）
1. 按下空格键角色执行跳跃动作
2. 跳跃高度为2米
3. 跳跃持续时间0.8秒
4. 角色在空中时不能二次跳跃
5. 落地时产生轻微震动效果

备注
- 颗粒度：0.5人日
- 技术债标签：TD（需要重写物理参数）
- 美术需求：ANI（跳跃动画）、SFX（跳跃音效）
